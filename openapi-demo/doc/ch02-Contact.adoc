=== 单位详情接口

接口地址：/info/getOrgInfo +
请求方式：GET +
*请求参数：*

[cols="<,<,<,<,<",options="header",]
|===
|名称 |类型 |是否必须 |默认值 |备注
|orgId |Long |必须 | |单位Id
|===

*返回参数：*
[cols="<,<,<",options="header",]
|===
|名称 |类型 |备注
|id |Long |单位id
|name |String |单位名称
|code |String |单位编码
|address |String |单位地址
|===
